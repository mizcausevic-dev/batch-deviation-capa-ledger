# Security

## Scope

This repository is a public proof surface and does not accept tenant secrets, protected health information, or production biotech records.

## Data posture

- synthetic batch deviation and CAPA evidence only
- no patient, clinician, or proprietary biotech secrets
- no live instrument credentials
- no claim of CLIA, GxP, FDA, HIPAA, or clinical compliance

## Reporting

If you identify a security issue in the code or Pages surface, open a private security advisory or contact the maintainer directly before public disclosure.
